TinyWP

Small WordPress Installation Program

Category Thumbnail

Adds shortcode `[catthumb]` to embed a thumbnail image for a category.

Gallery Excerpt

Replace the excerpt for posts with the post type 'Gallery' to a row of images from the post.

Offsite Post

Make a post redirect to an external URL

Embed PDF

Adds pseudo oembed support for PDF documents

Export to InDesign

Export a post as Adobe TaggedText for import to InDesign

Postlist Shortcode

Adds shortcode `[postlist]` for embedding a list of posts into a page

Better YouTube Embeds

Embeds YouTube videos in a way compatible with Desktop, iPhone, iPad, Android and other mobile browsers.

Embed RSS

Adds pseudo oembed support for RSS feeds

Kill Howdy

Changes the text **Howdy** in the admin interface to a different greeting.
